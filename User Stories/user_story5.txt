User logs in to application
User starts game, and is prompted with artwork
User guesses an incorrect location (ex. France for Belgian art)
User is told their guess is incorrect, that they have 2 tries remaining
User guess an incorrect location (ex. Switzerland)
User is told their guess is incorrect, that they have 1 try remaining
User guesses correctly
User is told their guess is correct, receives 1/3 points for this round
